# SMAart View — Embed Guide (for Real Estate Websites)

This guide explains how to add the SMAart View widget to a real estate website.

No technical background is required.

---

## What the widget does

Once installed, SMAart View shows a small on-page card with:

- recent listing views
- the listing that currently attracts attention
- the area generating interest

The widget is visible only to you and your team.
Visitors are not tracked or identified.

---

## Installation (2 minutes)

1. Copy the embed code you received.
2. Paste it into your website HTML,
   just before the closing `</body>` tag.
3. Save and refresh the page.

That’s it.

The widget will appear in the bottom-right corner.

---

## Example embed code

```html
<script
  src="https://pulse-api.anna-fokidou.workers.dev/widget.js"
  data-site-id="your-site-id">
</script>

Will this affect my website?

No.

No cookies are added

No page speed impact

No interference with existing analytics

No GDPR consent banner required

If the widget is unavailable, the page loads normally.

Can I remove it?

Yes.

Simply delete the script tag.
No residual data or configuration remains.
